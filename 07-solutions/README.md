  - 예: `programmers/lv2_target_number.cpp`
- **LeetCode**: `leetcode/{문제번호(4자리)}_{영문_문제명}.cpp`
  - 예: `leetcode/0200_number_of_islands.cpp`

### 풀이 작성 시 권장 사항

1. **주석에 풀이 요약을 반드시 작성합니다.** 나중에 복습할 때 빠르게 핵심을 파악할 수 있습니다.

## 학습 순서 제안

```
1단계: 기본 자료구조 (스택, 큐, 해시) → 정렬
2단계: BFS/DFS → 완전탐색/백트래킹
3단계: DP 기초 (1차원 → 2차원 → 배낭)
4단계: 그래프 (다익스트라, 플로이드, MST, 위상정렬)
5단계: 이분탐색 → 투포인터 → 슬라이딩 윈도우
6단계: 문자열 (KMP, 트라이)
7단계: 세그먼트 트리 → 고급 DP
```

각 단계에서 기본 문제를 먼저 풀고, 유형이 익숙해지면 중급/고급 문제로 넘어가는 것을 권장합니다.
